> This is a project for the Alkemy Challenge. It is a restaurant website where you can search for dishes and make your menu.

### 🏠 [Homepage](https://github.com/Z3belek/alkemy-restaurant)

## Install

```sh
npm install
```

## Usage

```sh
ng serve --open
```

## Run tests

```sh
ng test
```
